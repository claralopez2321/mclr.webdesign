Sobre el proyecto

Diseñé y desarrollé un sitio web utilizando HTML y CSS para el lanzamiento de Bok Seguros, enfocándome en crear una presencia digital sólida y profesional. El objetivo principal era dar a conocer la marca, transmitir confianza y facilitar el acceso a información clave sobre los servicios ofrecidos.

El proyecto incluyó un diseño moderno, adaptado a dispositivos móviles, con secciones claras para pólizas, contacto y sobre la empresa. Este trabajo permitió a la aseguradora posicionarse en línea y conectar con nuevos clientes desde el inicio de sus operaciones.




Características principales

Desarrollo de identidad visual

Se definió la identidad visual de Bok Seguros, estableciendo una estética profesional, moderna y coherente para transmitir confianza desde el primer contacto. Esto incluyó paleta cromática, tipografías y lineamientos gráficos aplicados en todo el sitio.

Sitio web a medida con HTML y CSS

El proyecto fue diseñado y codificado desde cero, garantizando una estructura limpia, rendimiento óptimo y una estética completamente alineada con el branding de la aseguradora.

Diseño moderno y responsive

La interfaz se adaptó cuidadosamente para ofrecer una experiencia fluida en móviles, tablets y escritorio, asegurando accesibilidad y claridad para todo tipo de usuarios.

Organización clara de contenidos

El sitio estructura la información de forma simple y directa, con secciones específicas para pólizas, contacto y datos institucionales, facilitando la consulta rápida y mejorando la conversión de potenciales clientes.
