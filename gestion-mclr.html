Sobre el proyecto

Desarrollé una aplicación de gestión personal diseñada para optimizar la organización diaria de trabajos y proyectos. La app permite gestionar tareas de manera eficiente, asignar pendientes, controlar totales y precios, y mantener toda la información vinculada para un acceso rápido y ordenado.
El proyecto fue implementado utilizando HTML, CSS y JavaScript, aprovechando estas tecnologías para ampliar las funcionalidades del sistema y ofrecer una experiencia intuitiva y fluida.
Actualmente, utilizo la aplicación de manera constante para administrar mis pedidos de diseño y desarrollo web como freelancer, lo que me permite ahorrar tiempo, mantener un mejor control sobre mis proyectos y mejorar la eficiencia en mi trabajo diario.

Características principales

Gestión de tareas eficiente

La app permite crear, organizar y priorizar tareas diarias, ofreciendo un control claro de los pendientes y su progreso.

Asignación de pendientes

Se pueden asignar tareas a diferentes proyectos o clientes, asegurando un seguimiento preciso de cada responsabilidad.

Control de totales y precios

La app registra costos y presupuestos asociados a cada tarea o proyecto, facilitando la gestión financiera y la planificación de recursos.

Acceso rápido y ordenado

Todo el contenido está vinculado y accesible de forma intuitiva, optimizando el flujo de trabajo y aumentando la productividad diaria.
