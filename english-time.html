Sobre el proyecto

Una profesora de inglés enfrentaba dificultades para organizar y distribuir el material de estudio entre sus alumnos. El envío de clases, tareas y correcciones se realizaba de forma manual a través de distintos canales, lo que generaba pérdida de tiempo, desorden en la información y complicaciones para hacer el seguimiento individual de cada estudiante.

Para resolver esta necesidad, desarrollé una plataforma web educativa con acceso personalizado para alumnos y un panel de administración para la docente. Desde este entorno puede crear perfiles de estudiantes, asignar clases en PDF, gestionar tareas específicas para cada alumno y centralizar toda la información académica en un único lugar.

El proyecto fue desarrollado utilizando HTML, CSS y JavaScript, con una interfaz responsive y una identidad visual personalizada. Además, se implementó en Firebase Hosting y se dejó preparado para una futura integración con Firestore y Storage, permitiendo evolucionar hacia una solución completa para la gestión, entrega y corrección de actividades educativas.

Características principales

Panel de administración

Permite a la docente crear perfiles de estudiantes, centralizar la información académica y gestionar tareas específicas para cada alumno desde un solo lugar.

Distribución de material

Acceso personalizado para que los alumnos visualicen y descarguen sus clases en formato PDF de manera ordenada y sin pérdidas de información.

Diseño moderno y responsive

Desarrollado con tecnologías web estándar (HTML, CSS y JavaScript) y una interfaz adaptada a cualquier dispositivo con identidad visual propia.

Despliegue y escalabilidad

Implementado en Firebase Hosting y preparado para una futura integración con Firestore y Storage para la gestión avanzada de actividades.
